# 📊 Sales Data Exploratory Data Analysis (EDA)

## Project Overview

This project focuses on **Exploratory Data Analysis (EDA)** of a sales dataset using Python. The main goal is to analyze sales performance, identify business trends, and extract meaningful insights through data visualization.

The project demonstrates data cleaning, statistical analysis, and interactive visualizations using powerful Python libraries.

---

## 🎯 Objectives

* Understand the structure of sales data
* Clean and preprocess raw data
* Analyze revenue, discounts, quantity, and product categories
* Discover hidden patterns and trends
* Create attractive visualizations for better business insights

---

## 🛠️ Technologies Used

* **Python**
* **Pandas** — Data cleaning & analysis
* **Matplotlib** — Static visualizations
* **Seaborn** — Advanced statistical plots
* **Plotly** — Interactive charts

---

## 📂 Dataset Features

The dataset contains information such as:

* Order ID
* Order Date
* Product Name
* Category
* Quantity Sold
* Price
* Discount
* Revenue

---

## 📈 Visualizations Included

This project includes multiple attractive charts such as:

* Revenue Distribution Histogram
* Revenue by Category Bar Chart
* Top Products Analysis
* Discount vs Revenue Scatter Plot
* Correlation Heatmap
* Revenue Trend Over Time
* Box Plot
* Donut Chart
* Treemap
* Sunburst Chart
* Bubble Chart
* 3D Scatter Plot

---

## 🔍 Key Insights

* Certain product categories generated significantly higher revenue.
* Higher discounts did not always increase sales.
* Revenue showed a strong correlation with price and quantity sold.
* Some products consistently outperformed others.
